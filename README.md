# Only Friends · Mission Control

Static web dashboard for The Only Friends Podcast. `index.html` reads `data.json` on load + every 90s auto-refresh. The `of-dashboard-snapshot` Cowork cron regenerates `data.json` every 5 minutes.

## Files

- `index.html` — the dashboard (self-contained, no build step)
- `data.json` — live pipeline state, regenerated by the snapshot cron
- `README.md` — this file

## Deploy options (pick one)

### A. Cloudflare Tunnel (fastest — public URL in 30 seconds)

```sh
# one-time install
brew install cloudflare/cloudflare/cloudflared

# start the local server in one terminal
cd "/Users/mjb11/Documents/Claude/Projects/Media Company Infrastructure/web"
python3 -m http.server 8080

# in another terminal, get a public URL
cloudflared tunnel --url http://localhost:8080
# → prints something like https://random-words.trycloudflare.com
```

The tunnel URL is ephemeral but works immediately. For a stable subdomain (`of.yourdomain.com`), create a named tunnel via Cloudflare Zero Trust.

### B. GitHub Pages (permanent URL, free)

```sh
cd "/Users/mjb11/Documents/Claude/Projects/Media Company Infrastructure/web"
git init && git add . && git commit -m "mission control v1"
gh repo create only-friends-mission-control --public --source=. --push
gh api -X POST /repos/$(gh repo view --json nameWithOwner -q .nameWithOwner)/pages \
  -f 'source[branch]=main' -f 'source[path]=/'
# URL: https://<you>.github.io/only-friends-mission-control/
```

To keep `data.json` fresh on GitHub Pages, extend the snapshot cron to `git commit + git push` the updated file (add a step 8 to the SKILL).

### C. Netlify / Vercel drag-and-drop

Drag the `web/` folder onto https://app.netlify.com/drop — you'll get a URL instantly. For live data, the snapshot cron needs to either push to the git repo backing the site or use the Netlify deploy API.

## Snapshot cron

Managed by Cowork:

- Task ID: `of-dashboard-snapshot`
- Schedule: `*/5 * * * *` (every 5 min, 24/7)
- Writes: `/Users/mjb11/Documents/Claude/Projects/Media Company Infrastructure/web/data.json`
- SKILL: `/Users/mjb11/Documents/Claude/Scheduled/of-dashboard-snapshot/SKILL.md`

To inspect or re-run manually, use the Scheduled tab in Cowork → `of-dashboard-snapshot` → Run now.
